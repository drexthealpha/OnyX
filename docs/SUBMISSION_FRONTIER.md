## 📝 Project Description
ONYX is a decentralized operating system for AI agents, built natively on Solana. It enables agents to be fully sovereign — owning their own wallets, paying for their own GPU compute via Nosana, signing transactions securely via dWallet MPC, and preserving total operational privacy via Umbra and Encrypt FHE.

Unlike traditional AI setups that rely on centralized SaaS middle-men, ONYX agents operate directly on-chain and on-device.

## 🚀 Key Features
- **Sovereign Intelligence:** Agents manage their own secrets and keys via dWallet and Ika.
- **Local-First Inference:** Integration with **QVAC** for offline-capable, private model execution.
- **Economic Autonomy:** Agents earn and spend USDt/SOL for compute (Nosana) and bandwidth.
- **Layered Architecture:** A modular stack from L0 Kernel to L9 Sovereignty, allowing for infinite expansion of agent capabilities.

## 🛠️ Tech Stack
- **Blockchain:** Solana (Mainnet-beta & Devnet)
- **Privacy:** Umbra (Stealth Addresses), Encrypt (FHE)
- **MPC/Signing:** dWallet, Ika
- **Compute:** Nosana, QVAC
- **Framework:** Next.js (Web), React Native (Mobile), Tauri (Desktop)
- **Infrastructure:** pnpm Monorepo, Turborepo
