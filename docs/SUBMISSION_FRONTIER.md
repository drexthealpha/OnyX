# Colosseum Frontier Hackathon — ONYX OS
**Grand Champion Track**

**Tagline:** Sovereign AI OS on Solana. 37 packages, 9 layers, zero operator cost.
**Team:** Onyx Core

## What it does
ONYX is a decentralized operating system for AI agents, built natively on Solana. It enables agents to be fully sovereign — owning their own wallets, paying for their own GPU compute via Nosana, signing transactions securely via dWallet MPC, and preserving total operational privacy via Umbra and Encrypt FHE. 

## Key Technical Insight
Agents must not have their private keys held in memory. ONYX integrates Ika and dWallet to ensure the agent's signing capability is decentralized and non-custodial.
