# Nosana & ElizaOS Track Submission — Decentralized GPU Compute

**Track:** Decentralized Compute & Agentic Orchestration
**Package:** `@onyx/gateway` · `@onyx/eliza-adapter`

## ⚡ The Compute Problem
Autonomous agents need reliable, high-performance GPU compute without being tethered to centralized cloud providers that can de-platform or censor them.

## 🤖 The ONYX Solution
We have containerized the **ONYX Gateway** and built a native **ElizaOS Adapter**, allowing agents to:
- **Nosana Deployment:** Deploy their own routing daemons to the Nosana decentralized GPU grid.
- **Autonomous Payments:** Agents pay for their own inference and compute jobs using NOS/USDt from their own wallets.
- **ElizaOS Synergy:** Bridging the ElizaOS ecosystem with ONYX's sovereign financial and memory layers.
